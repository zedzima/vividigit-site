Main Block with Sections

Unique feature: structured sections with anchor links

What Are Sections?

Sections are structured content blocks with titles - not free-form HTML. Use them when you need predictable, consistent formatting across multiple topics.

When to Use Main Block

Use Main when you have multiple distinct topics that need headings. Each section gets its own h3 and text block, making it easy to scan and navigate.

Anchor Links

The Main block supports anchor links - users can link directly to #main-demo in the URL. Useful for long pages with table of contents.

vs Text Block

Text block is for free-form content with columns. Main block is for structured, titled sections. Choose based on your content structure needs.
